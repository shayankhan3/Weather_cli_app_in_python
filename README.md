# 💻 SkyCast CLI - Console Weather Tool

A lightweight, Command-Line Interface (CLI) application that fetches and displays real-time weather information directly in your terminal. Built with a strong focus on clean backend logic, API integration, and efficient data parsing.

## 🚀 Features

*   **Instant Terminal Updates:** Get live temperature, humidity, and wind speed within seconds.
*   **Global Search:** Enter any city name via the console prompt to retrieve current weather metrics.
*   **ASCII Art (Optional):** Dynamic terminal text feedback based on current weather conditions.
*   **Robust Error Handling:** Smoothly handles invalid city names, network timeouts, and API errors without crashing.

## 🛠️ Tech Stack & Prerequisites

*   **Language:** Python 3.x
*   **Libraries Used:** `requests` (for API calls)
*   **API Provider:** OpenWeatherMap API
